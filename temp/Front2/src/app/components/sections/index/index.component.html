Funcionamiento:

Esta pagina web se centra en el concepto de crear un universo a base de Textos. Los usuarios podran leer todos los textos. Los usuarios que tengan cuenta ademas podran crear textos y valorar los de otros usuarios. Los textos con mas votos formaran parte del universo.

Para registrarte necesitas tener un nombre valido que no tenga caracteres o simbolos raros. Esto se debe a que este sera el nombre de tu personaje. Ademas para poder entrar tu nombre tendra que ser aceptado por un personaje ya registrado el cual sera tu padre o madre. Tu personaje tendra el apellido del padre o madre.
